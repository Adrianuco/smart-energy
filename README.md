# SmartEnergy

## Descripción del proyecto
SmartEnergy es una aplicación orientada a la optimización del consumo energético en entornos universitarios, específicamente en el uso de aires acondicionados dentro de las aulas.

El sistema permite monitorear el estado de los equipos en función de los horarios académicos, detectar posibles situaciones de desperdicio energético y generar alertas para el personal de soporte técnico. Además, incluye un módulo de reporte de incidencias y un dashboard con métricas de consumo estimado.

El objetivo principal es reducir costos operativos, mejorar la eficiencia en la gestión de recursos y contribuir a iniciativas de sostenibilidad como UAM Verde.
